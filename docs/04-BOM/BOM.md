## Overview
This is the bill of materials (BOM) section which describes all the components used in the design, their price, quantity, and link to product.
> **Note:** all products were found on digikey except for the stepper motor, pin headers and sockets, and resistors, as those were provided and did not need to be purchased.
> **Exception:** the test points were added in the final version of the PCB so no specific one was selected for the BOM.

You can download the BOM as a pdf [here](https://botilarm.github.io/04-BOM/sable_arm_BOM.pdf).
