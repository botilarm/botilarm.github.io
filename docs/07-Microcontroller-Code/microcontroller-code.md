## Overview
This code was used to verify subsystem functionality by converting an analog signal from an amplifier to a DC voltage, and displaying it, as well as sending a digital signal to verify that current is being sensed to the solenoid board. It features code for switching the display from energy usage to instantaneous power but does not yet feature code to convert from the analog to digital signal to show either of those two values.

The code as an mplabx project download is available [*here*](sg_power_monitor.zip).
